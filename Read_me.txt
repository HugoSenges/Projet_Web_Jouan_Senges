//Comment installer le jeu
1) Vous disposez d'un dossier avec plusieurs éléments dedans:
	- Un fichier web.sql
	- Un fichier JS
	- Deux fichiers HTML qui correspondent aux deux pages, l'une pour l'accueil et l'autre pour le jeu
	- Plusieurs fichiers PHP qui gèrent les appels à la base de données

## Sous Windows avec EasyPHP
2) Lancer EasyPHP et faire "OpenDashboard"
3) Une fois sur la page d'accueil, lancez les deux serveurs (HTTP et DATABASE)
4) Ajoutez un file directory correspondant au dossier fourni avec le chemin associé
5) Cliquez sur le bouton "open" de la ligne MySQL pour arriver sur la page de PHPMyAdmin
6) Créez une nouvelle base de donnée, et nommez-la "web"
7) Une fois cette base de donnée sélectionnée, allez dans l'onglet "Importer" puis cliquez sur le bouton "Open File" pour sélectionner le fichier web.sql mentionné au début
8) Si il n'y a pas d'erreur, fermez la page
9) Retournez sur le Dashboard de EasyPHP
10) Ouvrez la page HTML appelée index.html
11) Mettez le son de votre ordinateur pour ne passer à côté d'aucun indice!
